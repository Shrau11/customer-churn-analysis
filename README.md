# customer-churn-analysis

## Project Overview:
This project focuses on analyzing customer churn data to
identify key factors that influence customer attrition.
The objectives is to understand customer behavior and provide actionable insights to reduce churn.

## Problem Statement:
Customer churn is a major issue for subscription-based businesses.
The goal of this project is to:
 - Identify churn patterns
 - Analyze important features affecting churn
 - Provided business insights for retension strategies.

## Tools and Technologies Used:
 - Python
 - Pandas
 - Numpy
 - Matplotlib
 - Seaborn
 - Jupyter Notebook

## Exploratory Data Analysis (EDA)
 - Checked missing values
 - Performed data cleaning
 - Analyzed categorical and numerical features
 - Visualized churn distribution
 - Correlation analysis

## Key Insights:
 - Customer with montj-to-month contracts have higher churn rate.
 - High monthly charges increase churn probability.
 - Customer with longer tenure are less likely to churn.
 - Electronic check payment method shows higher churn rate

## Conclusion:
By understanding churn patterns, business can design targeted retension strategies such as loyalty
programs, long-term contracts, and improved customer services.
